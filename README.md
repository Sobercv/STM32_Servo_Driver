# STM32 双闭环直流伺服电机控制系统

## 📖 项目简介
本项目基于 STM32F103C8T6，实现对直流减速电机的高精度位置与速度控制。
- **控制算法**：串级 PID（位置环 + 速度环） + 三段式前馈补偿
- **通信方式**：支持 CAN 总线指令 / 串口调试 (VoFA+)
- **交互界面**：0.96寸 OLED 显示状态 + 按键多模式切换

## 🛠️ 硬件清单
- **主控**：STM32F103C8T6
- **电机**：JGA25-370B (减速比 1:103)
- **驱动**：TB6612FNG / AT8236
- **屏幕**：0.96 OLED (I2C)

## 🔌 接线说明
| STM32引脚 | 连接设备 | 功能 |
| :--- | :--- | :--- |
| PA0 | 驱动 PWMA | PWM输出 |
| PB12/13 | 驱动 AIN1/2 | 方向控制 |
| PA6/7 | 编码器 A/B | 测速 |
| PA1/2/3 | 按键 | 模式/加/减 |

## 🚀 快速开始
1. 使用 STM32CubeMX 打开 `.ioc` 文件生成工程。
2. 编译并下载代码。
3. 按下 PA3 按键启动电机，进入速度模式。
